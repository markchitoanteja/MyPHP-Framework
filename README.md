A lightweight, educational, and extensible **PHP MVC micro-framework** built from scratch.

## Requirements
- PHP 8.0+
- Apache (XAMPP/LAMP)
- MySQL/MariaDB

## Installation
1. Copy the framework into your web root (e.g. `htdocs/test`)
2. Enable `mod_rewrite`
3. Create `.env` in the root
4. Visit `http://localhost/test`
